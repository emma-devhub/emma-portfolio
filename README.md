# Emma Sun — AI Product Builder Portfolio

Personal portfolio site. Built with Next.js, TypeScript, and Tailwind CSS.

## Case Studies

- **[ChatGeneT](src/app/work/chatgenet/page.tsx)** — AI patient simulator for clinical training. Multi-turn dialogue system deployed across 30+ hospitals. Built the eval framework, dialogue taxonomy, and release gates.
- **[Valence](src/app/work/valence/page.tsx)** — Expert-agent marketplace. Built the teach-test-publish loop, owned front-end design, and hand-built 10+ live agents from scratch.
- **[Repurpose](src/app/work/repurpose/page.tsx)** — AI content redistribution CLI. Ingests long-form content and publishes platform-native drafts across X, LinkedIn, and Xiaohongshu with per-account persona config.
- **[Decoding the Beige Book](src/app/work/beigebook/page.tsx)** — LLM-powered macro risk research. End-to-end text-to-signal pipeline across 526 Fed Beige Books. ACM ICAIF 2025 oral presentation. Peak F1 0.89.

## Tech Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- TypeScript
- [Tailwind CSS](https://tailwindcss.com/)
- Playfair Display + Libre Franklin (Google Fonts)

## Run Locally

```bash
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000).
